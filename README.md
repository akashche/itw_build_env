Iced-Tea Web build environment
==============================

Windows build environment for ITW 1.8.

Usage:

    git clone https://github.com/akashche/itw_build_env.git
    cd itw_build_env
    git clone --branch=itw-1.8 https://github.com/ojdkbuild/lookaside_icedtea-web.git
    # copy jdk to "jdk" dir
    build.bat
